# RuCode 2025 AI Championship

Репозиторий содержит решения практических задач чемпионата [RuCode 2025](https://rucode.net/ai_champ/) по искусственному интеллекту.
Задачи предоставлены компаниями-партнёрами и ориентированы на применение методов машинного обучения и анализа данных.

## Требования
- python 3.11
- Пакеты:
  - numpy==2.3.3
  - pandas==2.3.3
  - scikit-learn==1.7.2
  - matplotlib==3.10.6
  - seaborn==0.13.2
  - notebook==7.4.5
