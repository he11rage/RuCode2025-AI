# BASE: Контент-мейкер (МТС) - RuCode AI 2025
### Score: 95.36 (F1-macro)

## Задача
Разработать модель машинного обучения для предсказания, досмотрит ли пользователь видео более чем на 50%. 
Решение было представлено в рамках соревнования RuCode AI 2025.

## Данные
- `interactions_train.csv` - обучающая выборка
- `users.csv` - данные пользователей
- `items.csv` - метаданные видео
- `interactions_public_test.csv` - тестовые данные
- `sample_sub_public_test_seed_0.csv` - пример предсказаний

## Ноутбук
- `EDA_and_Modeling` - EDA, препроцессинг и обучение модели

## Пайплайн

### EDA
- Рассмотрены распределения признаков медиаконтента и пользователей;
- Выявлены различия в поведении пользователей при просмотре фильмов и сериалов;
- Обнаружены выбросы в признаке длительность контена и нормализованы;

### Feature Engineering
- Заполнены пропуски
- Логарифмирование признака total_dur для нормализации распределения.
- Преобразование текстовых данных в числовой формат (age, income, sex, kids_flg/age, content type);
- Декомпозиция last_watch_dt на watch_hour, is_weekend, weekday;
- One-Hot-Encoding для топ-30 жанров/стран (покрывают 99.3% контента), остальные выделены в категорию 'Other';
- Добавлен признак по количеству ключевых слов;

### Modeling
- Baseline логистическая регрессия f1-macro **0.76**
- XGBoost classifier, для бинарной классификации целевого признака f1-macro **0.96**
- XGBoost regressor, для оценки целевого признака в интервале [0, 100] коэффициент детерминации = **0.96**

## Ключевые признаки
- Продолжительность контета;
- Типо контента (фильм или сериал);
- Мета-данные видеоконтента (страна производства, жанр).
