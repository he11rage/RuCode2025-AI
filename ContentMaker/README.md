# BASE: Контент-мейкер (МТС) — RuCode AI 2025

## Описание
Предсказание, будет ли пользователь досматривать видео >50% на основе истории просмотров, метаданных видео и демографических данных пользователей.

## Данные
- `interactions_train.csv` — обучающая выборка
- `users.csv` — данные пользователей
- `items.csv` — метаданные видео
- `interactions_public_test.csv` — тестовые данные
- `sample_sub_public_test_seed_0.csv` — пример предсказаний

## Скрипты
- `train.py` — обучение модели
- `predict.py` — предсказание на тестовом наборе, создаёт `.csv`

## Метрика
- F1-macro

## Библиотеки
Python 3.11 + `requirements.txt`

## Seed
- Использован seed 42 для воспроизводимости
